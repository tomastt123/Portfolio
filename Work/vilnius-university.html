Reimagined Vilnius University website

This project focused on the complete redesign of the Vilnius University website, modernizing its look and improving user experience. Built with React, HTML, CSS, and JavaScript, the new design delivers a responsive, user-friendly interface that enhances accessibility for both prospective and current students. The website efficiently organizes a large amount of information into a clean, easy-to-navigate layout, offering quick access to course details, campus news, and administrative resources. The updated design reflects the university's rich history while incorporating modern web design trends, giving visitors an intuitive experience across all devices.

Key features include:

  • Responsive design adaptable to all screen sizes
  • Modern layout for improved navigation and usability
  • Integration with React for enhanced performance and interactivity
  • Focus on accessibility to ensure easy use for all types of users
